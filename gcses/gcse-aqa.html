Searching & Sorting

Linear Search: Sequential scan; O(n).

Binary Search: Sorted list; halving approach; O(log n).

Bubble Sort: Repeated swaps; O(n²).

Merge Sort: Divide and merge; O(n log n).

// Binary search pseudocode
low ← 0; high ← len(list)-1
WHILE low ≤ high
  mid ← (low + high) DIV 2
  IF list[mid] = target THEN RETURN mid
  ELSE IF list[mid] < target THEN low ← mid + 1
  ELSE high ← mid - 1
ENDWHILE
RETURN -1
