AQA GCSE Computer Science Topics

Students should understand the use of decomposition, abstraction, and algorithmic thinking. They should be able to interpret and produce flowcharts and pseudocode.

  • Decomposition & Abstraction: Breaking problems into smaller parts and focusing on the essentials.
  • Flowcharts/Pseudocode: Represent algorithms graphically or in plain English.
  • Searching/Sorting: Bubble, Merge, Insertion sort; Linear & Binary search.

Core programming principles: students must understand how to write, test, and refine code.

  • Data Types & Variables: integers, strings, booleans.
  • Operators: arithmetic, Boolean logic.
  • Input/Output: reading user input and displaying output.
  • Control Structures: IF, FOR, WHILE loops, nested selection/iteration.
  • Subroutines: Functions & procedures to modularize code.

Explore how data is stored and processed:

  • Number Bases: binary (base-2), hexadecimal (base-16), conversions to/from decimal :contentReference[oaicite:2]{index=2}.
  • Units of Data: bits/bytes; kilo/mega/giga/tera prefixes.
  • Binary Arithmetic: addition and bit-shifts.
  • Character Encoding: ASCII & Unicode.
  • Image/Sound Representation: pixels, colour depth; sampling rate, bit resolution.
  • Compression: Huffman coding, run-length encoding.

Understand the architecture and performance of computer hardware:

  • CPU & Fetch-Execute: registers, ALU, CU, cache, Von Neumann model :contentReference[oaicite:3]{index=3}.
  • Memory: RAM, ROM, cache – volatile vs. non-volatile.
  • Secondary Storage: SSD, HDD, optical – pros/cons.
  • Embedded Systems: special-purpose computers.

Performance Factors: clock speed, cores, cache size.

Key network concepts:

  • LAN/WAN and common topologies.
  • Networking Hardware & Protocols: switches, routers, Ethernet, TCP/IP.
  • Network Security: firewalls, anti-malware, encryption, types of attack. :contentReference[oaicite:4]{index=4}

Protecting systems and data:

  • Malware types: viruses, worms, trojans, ransomware.
  • Social engineering: phishing, pharming.
  • Prevention: passwords, authentication, anti-malware, encryption, software updates.

Managing and querying data:

  • Relational databases: tables linked by keys.
  • SQL: SELECT, INSERT, UPDATE, DELETE queries.

How technology impacts society:

  • Privacy concerns, hacking, surveillance.
  • Legislation: Data Protection Act, Computer Misuse Act.
  • Ethical issues: AI, digital divide.
  • Environmental impacts: e-waste, energy use of data centres. :contentReference[oaicite:5]{index=5}
